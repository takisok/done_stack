# ADR 0001: ブラウザ内保存を使う

## Status

Accepted

## Context

できたスタンプ💮 は個人用のローカルブラウザアプリです。
サーバーを用意せずに、すぐ使えることを優先します。

## Decision

Done データはブラウザ内に保存します。
優先保存先は IndexedDB とし、使えない場合のみ localStorage にフォールバックします。

## Consequences

- サーバーなしで利用できます。
- ブラウザやプロファイルを変えるとデータは共有されません。
- バックアップや移行は CSV エクスポート / インポートで行います。
