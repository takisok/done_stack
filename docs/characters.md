# キャラクター設計

## メイドさん

現在アプリで参照する画像:

- `img/maid/maid-idle.png`
- `img/maid/maid-thinking.png`
- `img/maid/maid-talk-1.png`
- `img/maid/maid-talk-2.png`

状態:

- 通常時: `maid-idle.png`
- LLM 待機中: `maid-thinking.png`
- 発話中: `maid-talk-1.png` と `maid-talk-2.png` の差分で口パク

LLM 待機中の絵:

- 目をつぶる
- 首をかしげる
- 人差し指を頬に当てる
- 思考丸は左側に 2 つ
- 吹き出しの尖りは非表示

発話方針:

- 入力テキストを引用しない
- 入力テキストを復唱しない
- 入力テキストを言い換えない
- 感想や応援だけを話す

## マスコット

現アプリで参照する画像:

- `img/mascot/mascot.png`

構想:

- 名前案: クラム
- 白くて丸い、もちもちしたマスコット
- 小さな案内役の雰囲気を持つ
- 既存キャラクターに直接似せない
- クッキークリッカーのオマージュとして、クッキー要素を持たせる
- クトゥルー要素は普段は目立たせない

クトゥルー要素の隠し場所案:

- 口の奥
- 体の底面
- しっぽの毛の奥
- 特殊状態で一瞬だけ見える場所

役割:

- メイドさんは明るく褒める役です。
- マスコットは淡々と観測する役です。

試作用に生成した画像は、採用が決まるまでアプリ参照に使いません。
採用する場合は、最終画像名を決めて `index.html` または `js/app.js` から参照します。
