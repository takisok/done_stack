# できたスタンプ💮 仕様

## 概要

できたスタンプ💮 は、達成したことを短いテキストで記録し、件数や履歴を見ながら積み上げを確認するローカルブラウザアプリです。

## 画面

### トップページ

- 上部は黒いヘッダーです。
- LLM 接続状態は黒いヘッダーの中ではなく、ヘッダー直下の白または薄い背景側に表示します。
- 中央の達成数は、数字を上、`達成数` ラベルを下に配置します。
- 入力欄にテキストを入力し、Done ボタンで記録します。
- Done リストは 20 件ごとにページネーションします。
- ページネーションがあるため、リスト部分の内部スクロールバーは使いません。
- 記録は編集・削除できます。

### 活動ログページ

- 表示の基本単位は月です。
- 年間の草表示は、対象月を選ぶための月ナビとして扱います。
- 草をクリックすると、その日単体ではなく、その日を含む月に移動します。
- 表示中の月に含まれる草マスは軽く強調します。
- `この月の記録` は、月内の Done を日付ごとにグループ表示します。
- `この月の傾向` には、カテゴリ、よく出た言葉、月間ハイライトを表示します。

## キャラクター反応

- 記録時、メイドさんが感想や応援を話します。
- LLM 接続中は LLM でセリフを生成します。
- LLM 未接続時は `js/speech.js` の静的セリフを使います。
- メイドさんは入力文を引用・復唱・言い換えしません。
- LLM 待機中は考えている立ち絵を表示し、吹き出しの尖りを消します。
- セリフ表示中は、差分画像で口パクを行います。

## 音

- 文字入力時にカチャカチャ音を鳴らします。
- 日本語入力中の IME 変換時にも入力音を鳴らします。
- Done ボタン押下時に「ッターン！」音を鳴らします。

## 活動傾向

- よく出た言葉は `Intl.Segmenter('ja-JP', { granularity: 'word' })` を優先して単語単位で集計します。
- 月間ハイライトは、現状では「本文が長い Done を優先し、同じ長さなら新しいものを優先」する仮実装です。
